Descrição dos projetos:

"Rua sem Saída" - A caixa preta de Villém Flusser.

A criação desse site foi feita na cadeira "Introdução a Informática" do curso de Arte e Mídia da UFCG. O conceito gira em torno de simular uma prisão para o usuário, onde a opção de "sair" do site não é de fato uma saída.

As tecnologias usadas foram: HTML, CSS e JavaScript.

"TECHCONNECT 2025 - Conectando ideias, criando o futuro"

A proposta deste trabalho veio na cadeira de PROGRAMAR WEB FRONT-END. O evento TECHCONNECT é fictício, e a sua página conta com: PROGRAMAÇÃO, PALESTRANTES, CONTATO, RECORDAÇÕES, etc. O site também possui o modo escuro.

Tecnologias usadas foram: HTML, CSS e JavaScript (para o modo escuro).

Projeto Centelhas do Cristo

O projeto "Centelhas do Cristo" é uma ação social real, que distribui marmitas pela cidade de Campina Grande aos sábados. Construímos um site para ajudar a instituição a divulgar sua ação.

Tecnologias usadas foram: HTML e CSS.
